```js
{
   "statusCode": 429,
   "message": "Rate limit is exceeded. Try again in X seconds."
}
```

​ 5분당 요청 8개의 제한이 있는데 이를 넘은 **잦은 요청으로 인한 에러**

요청에 대한 카운트는 endpoint, ip, 배포키, 클라이언트 고유 아이디로 계산됨.

해당 에러가 뜨면 **시간이 좀 지나고 다시** 하면 잘 된다
